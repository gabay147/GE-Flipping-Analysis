# GE-Flipping-Analysis

by Felix Gabay

[GE-Flipping-Analysis](github.com/gabay147/GE-Flipping-Analysis) is a data 
analysis project exploring daily GE prices from 2021 to 2026 that intends to 
inform users on ideal flipping strategies on the Grand Exchange in the popular 
MMORPG Old School Runescape.

*Flipping* is a form of commodities trading in the game's player-run
micro-economy, wherein players trade materials on the Grand Exchange (GE for 
short; the game's player-to-player goods auction house) in a "buy low, sell 
high" strategy for incremental profit. I play OSRS fairly casually so I never 
dipped my toes into flipping. After graduating and struggling to find a job, I 
developed this project as a means to keep my data analysis skills sharp, explore 
new tools, and apply my skills to a project that impacts my daily life. 
Ironically, I stopped playing as I started this project.

## How to use this project

Honestly, I'm not sure that this project will have much use to the average
gamer. If you decide to explore some of the statistical analyses or code, you
will (hopefully) find insight in statistics, data communication, documentation,
and machine learning.

As of 1 July 2026, this project is in its infancy, and the potential uses are 
simply speculative. I have a lot of ideas and time will tell where this goes.

## Data Provenance

The data set used in this project is the [OSRS GE Snapshot](https://www.kaggle.com/datasets/felixgabay/osrs-ge-prices-snapshot-june-27-2026).
I used the [OSRS Wiki Pricing API](https://oldschool.runescape.wiki/w/RuneScape:Real-time_Prices) to build this dataset. 
Code related to building this dataset can be found at the [GE-Data-Collector](https://github.com/gabay147/GE-Data-Collector).
